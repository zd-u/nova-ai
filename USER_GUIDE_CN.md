## 问题 2：电脑关机后还能用吗？

### 现状解释

**答案是：不能。** 如果我的电脑关机或断网，你的手机上的 Nova 应用会立刻无法使用。

### 为什么会这样？

```
你的手机 ← 网络连接 → 我的电脑（沙盒环境）
                      ↓
                   后端服务器（API）
                      ↓
                   大模型 API（DeepSeek/OpenAI 等）
```

**目前的问题：**
- ❌ 后端服务器**仍在我的本地电脑上运行**
- ❌ 没有部署到云端（如 Render、Vercel）
- ❌ 如果我的电脑关机，整个链路断裂
- ❌ 你的手机无法连接到后端 API
- ❌ 聊天功能完全瘫痪

### 对比：部署到云端后会怎样？

```
你的手机 ← 网络连接 → 云端服务器（Render/Vercel）
                      ↓
                   后端服务器（API）
                      ↓
                   大模型 API（DeepSeek/OpenAI 等）
```

**部署到云端后：**
- ✅ 后端服务器运行在云端（24/7 在线）
- ✅ 我的电脑可以关机
- ✅ 你的手机随时可以聊天
- ✅ 完全独立，不依赖我的电脑

## 💡 重要提示

### 关于 API Key

在整个过程中，**你需要自己的大模型 API Key**（如 DeepSeek、OpenAI 等）。

**为什么需要？**
- Nova 本身不提供 LLM 服务
- 需要调用第三方大模型 API 来生成回复
- API Key 是你和大模型提供商之间的凭证

**怎么获取？**
- 去 https://platform.deepseek.com 注册账户
- 或去 https://platform.openai.com 注册账户
- 获取你的 API Key
- 在应用的 Settings 中填入 API Key

**费用？**
- DeepSeek：免费额度充足（约 5 美元免费额度）
- OpenAI：按使用量计费（通常每月 5-20 美元）
- 其他平台：各有不同

- ✅ 你的聊天记录完全保存在手机本地
- ✅ 不会上传到我的服务器
- ✅ 你的 API Key 只在手机上存储
- ✅ 完全安全和隐私

## 📞 常见问题

### Q1：如果我不想部署到云端，能直接生成 APK 吗？

**A：** 可以，但 APK 只能在你的电脑开机时使用。一旦电脑关机，应用就无法聊天。这不是理想的使用体验。

### Q2：部署到云端需要付费吗？

**A：** Render 提供免费层，完全免费。如果你的应用使用量很大，可能需要升级到付费计划（通常每月 7-10 美元）。

### Q3：生成 APK 需要付费吗？

**A：** Expo 提供免费的 APK 生成服务。完全免费。

### Q4：我可以把 APK 分享给朋友吗？

**A：** 可以。朋友可以直接安装你的 APK 文件。但他们需要在 Settings 中填入自己的大模型 API Key 才能聊天。

### Q5：APK 大小是多少？

**A：** 大约 50-100MB，取决于你的手机型号和系统版本。

### Q6：支持 iOS 吗？

**A：** 目前我们只生成 Android APK。iOS 需要额外的步骤（需要 Mac 电脑和 Apple Developer 账户）。
